# Blueprint

Declarative domain modeling language for building APIs with invariant rules.

## What is Blueprint?

Blueprint is a modeling language (DSL) designed to describe domain rules,
aggregates, and APIs in a declarative way.

It generates production-ready C# projects following:
- DDD
- Clean Architecture
- CQRS
- Azure-first infrastructure

## Philosophy

- Explicit over magic
- Domain rules first
- Generated code is readable
- Infrastructure is replaceable
